## The organising principle: two doors

Most of the audience is **problem-UNAWARE** (§3), so the site must *create* demand as well as *capture* it:

```
DOOR A — DEMAND CREATION  (unaware majority; from social / Tate / SMMA / cold)
   hook → /scorecard → email + instant RAG result → book a call
                                   ↘ Simon sends the full one-pager (manual, for now)

DOOR B — DEMAND CAPTURE   (aware minority; from Google search)
   /accountants-for-marketing-agencies (deep SEO page) → proof → book a call

BOTH converge on:  Book a discovery call  (Google Calendar link)
```
